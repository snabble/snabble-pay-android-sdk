enum MandateState : Enum<MandateState>

Represents the state of a Mandate.

If a new mandate has been created but has been neither accepted nor declined, the state is PENDING. After it has been accepted or declined it is in its final state ACCEPTED or DECLINED. If there is no mandate for the account the state is MISSING.
