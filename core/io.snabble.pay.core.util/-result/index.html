Result

sealed class Result<out T>

Result wrapper for return value.

Returns the requested object or null on Success or an exception, with the responding PayError on Failure. The return value can be accessed via two ways: type check: Success or Failure properties isSuccess or isFailure, receive the object via the corresponding functions getOrNull and exceptionOrNull
